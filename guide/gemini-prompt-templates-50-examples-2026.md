# Gemini提示词大全2026：50个高效Prompt模板让AI输出质量翻倍

**更新时间：2026年4月** | 全文约5000字 | 阅读时间：15分钟

> 📌 同样是用Gemini，为什么有人能得到精准有用的回答，有人却只能得到泛泛而谈的内容？秘密就在于提示词（Prompt）。

---

## 为什么提示词这么重要？

Gemini 3.1 Pro是目前最强大的AI模型之一，但它的输出质量**90%取决于你怎么问**。

一个好的提示词和一个随意的提问，结果可能天差地别：

| 提问方式 | 示例 | 输出质量 |
|----------|------|:---:|
| 随意提问 | "帮我写个文案" | ⭐⭐ |
| 具体提问 | "帮我写一个针对25-35岁女性的护肤品种草文案，产品是保湿面霜，800字，小红书风格" | ⭐⭐⭐⭐⭐ |

下面我会给你50个经过验证的Prompt模板，直接复制粘贴就能用。

---

## 提示词的4个黄金法则

在看具体模板之前，先掌握这4条通用原则：

### 法则1：明确角色

告诉Gemini它是谁，它就会用那个身份来回答。

```
你是一位有10年经验的产品经理，擅长B2B SaaS产品。
```

### 法则2：具体任务

把"帮我写"变成"按照XX格式/风格/要求写XX内容"。

### 法则3：给出约束

字数、格式、语气、受众——约束越清晰，输出越精准。

### 法则4：提供示例

如果你有理想输出的参考样本，直接给Gemini看，效果倍增。

## 一、写作类Prompt（10个）

### 1. 公众号爆款标题

```
你是一位资深新媒体运营，擅长写10W+爆款标题。请为以下文章主题生成10个标题：
主题：[你的主题]
要求：
- 包含数字的标题至少3个
- 包含悬念/反转的标题至少3个
- 每个标题不超过25字
- 适合微信公众号传播
```

### 2. 小红书种草笔记

```
你是一位小红书博主，粉丝5万，定位[领域]。请写一篇种草笔记：
产品：[产品名称]
要求：
- 标题要有emoji，吸引点击
- 正文口语化，像和朋友聊天
- 800-1200字
- 包含3个使用场景
- 最后有互动引导
```

### 3. 专业分析报告

```
你是一位[行业]分析师，请写一份关于[主题]的分析报告。
结构要求：
1. 行业背景（200字）
2. 现状分析（500字，配数据）
3. 核心问题（300字）
4. 趋势预测（300字）
5. 建议方案（200字）
语气：专业、客观、有数据支撑
```

### 4. 英文商务邮件

```
请帮我写一封英文商务邮件。
场景：[具体场景，如：跟进客户订单进度]
发件人角色：[你的职位]
收件人：[对方职位]
关键信息：[需要传达的重点]
语气：专业礼貌但不过于正式
长度：150-200词
```

### 5. 视频脚本

```
请为一个[时长]分钟的[平台]视频写脚本。
主题：[主题]
受众：[目标受众]
格式：
- 开头钩子（前5秒抓住注意力）
- 内容主体（分3-5个要点）
- 结尾CTA
- 标注画面建议和字幕文案
```

### 6. SEO文章

```
请写一篇SEO优化文章。
核心关键词：[关键词]
长尾关键词：[3-5个长尾词]
要求：
- 标题包含核心关键词
- H2/H3标题自然包含关键词
- 正文2000-3000字
- 关键词密度1%-2%
- 每300字一个小标题
- 开头前100字包含核心关键词
```

### 7. 产品描述文案

```
为以下产品写一段吸引人的描述文案：
产品名：[名称]
类别：[类别]
核心卖点：[3个卖点]
目标用户：[用户画像]
使用场景：[2-3个场景]
字数：300字
风格：[简约大气/活泼有趣/专业权威]
```

### 8. 会议纪要

```
请根据以下会议内容整理一份结构化的会议纪要：
[粘贴会议内容/录音转文字]
格式要求：
- 会议主题
- 参会人
- 核心讨论要点（分条列出）
- 决策事项
- 待办任务（标注负责人和截止日期）
```

### 9. 朋友圈文案

```
帮我写5条朋友圈文案。
场景：[旅行/美食/读书/工作感悟]
风格：[文艺/幽默/治愈/高级感]
要求：每条不超过100字，配emoji，不要太鸡汤
```

### 10. 学术论文摘要

```
请帮我改写/润色以下论文摘要：
原文：[粘贴原文]
要求：
- 保持学术语言风格
- 结构：背景→方法→结果→结论
- 200-300词（英文）
- 减少被动语态的使用
```

```
- 包含错误处理
```

```
3. 安全隐患
```

### 13. 代码解释


### 14. 正则表达式

```
[描述表结构或粘贴CREATE TABLE语句]
```

## 提升Prompt效果的5个高级技巧

### 技巧1：链式思考（Chain of Thought）

在Prompt中加入"请一步一步思考"，可以显著提升Gemini在复杂推理任务中的准确率。

### 技巧2：Few-shot示例

给出2-3个输入-输出的示例，让Gemini理解你想要的格式和风格。

### 技巧3：自我评估

在Prompt末尾加上"完成后，请自我评估输出的质量（1-10分），并指出可以改进的地方"。

### 技巧4：迭代优化

第一次输出不满意？别重写Prompt，直接追问"请在以下方面改进：[具体要求]"。

### 技巧5：角色叠加

给Gemini多个角色，让它从不同视角分析同一问题：

```
请分别以CEO、CTO和用户的视角评价这个产品方案。
```

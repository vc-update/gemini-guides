## 第三步：认识Gemini 3.1 Pro的4大核心能力

Gemini 3.1 Pro不只是"另一个聊天机器人"。它有4个真正区别于其他模型的核心能力：

### 🧠 能力1：100万Token超长上下文

这是Gemini的**杀手级优势**。100万tokens大约等于：
- 一本75万字的小说
- 30小时的会议录音转文字
- 一整个中型代码仓库

**实际用途：**
- 上传一份200页的合同PDF，让AI逐条分析风险点
- 丢进一整本教材，让AI生成章节摘要+思维导图
- 把整个项目代码库喂给它，让它做全局代码审查

这种"一次性吃下大量信息"的能力，GPT-5.4（12.8万）和Claude 4.6（20万）都做不到。

### 🔬 能力2：Deep Think深度推理

开启Deep Think后，Gemini会展示完整的推理链——先分解问题、假设验证、再得出结论。特别适合：

- 数学证明题
- 编程难题debug
- 商业策略分析
- 科研实验设计

### 🎨 能力3：原生多模态

Gemini从训练阶段就是多模态的——不是后期"加上"的图片理解，而是原生就能处理：

- **图片**：看图分析、OCR文字识别、图表数据提取
- **音频**：语音转文字、会议录音分析
- **视频**：视频内容理解、关键帧提取
- **代码**：读代码截图、分析报错界面

### 🌐 能力4：实时信息能力

Gemini背靠Google搜索生态，能获取最新信息。问它"今天的新闻""最新的某某技术进展"，它给出的信息比其他模型更新、更准确。

## 第四步：8大实战场景——手把手教你用Gemini

以下8个场景覆盖了Gemini最常用的方向。每个场景都配有**直接可复制的Prompt模板**。

### 场景1：长文档分析（Gemini独有优势）

这是Gemini最值得用的场景——利用100万token的上下文窗口：

```
请分析以下合同文件，重点关注：
1. 对甲方不利的条款（列出条款编号+风险等级）
2. 模糊表述可能导致的争议点
3. 缺失的常见保护性条款
4. 给出修改建议

[粘贴完整合同内容]
```

**适合：** 法务合同审查、投标文件分析、论文文献综述、代码库审计

### 场景2：学术研究助手

```
我正在研究"[研究课题]"。请帮我：
1. 梳理该领域2024-2026年的核心进展（5-8篇关键论文）
2. 归纳当前的主流研究范式和争议焦点
3. 识别尚未解决的开放问题
4. 建议3个有潜力的研究方向
要求：每个观点注明依据，区分"已有共识"和"仍有争议"的内容。
```

### 场景3：深度推理（开启Deep Think）

```
[建议开启Deep Think模式]

公司Q1营收同比增长15%，但净利润下降8%。同期：
- 原材料成本上涨22%
- 新产品线占营收比从5%升至18%
- 员工数增长30%
- 营销费用增长45%

请分析：利润下降的主要驱动因素是什么？哪些是短期投入，哪些是结构性问题？
给出6个月和12个月的改善建议。
```

### 场景4：代码生成与调试

```
用Python实现一个带缓存的异步Web爬虫：
要求：
- 支持并发控制（最大10个并发请求）
- 自动重试（3次，指数退避）
- 结果缓存到SQLite
- 支持自定义请求头和代理
- 完善的日志记录
- 包含类型注解和docstring
```

### 场景5：多模态图片分析

上传一张图片，然后输入：

```
请分析这张图片，提供：
1. 图中包含的所有文字内容（OCR）
2. 数据图表的关键数据点和趋势
3. 图中物体/人物/场景的描述
4. 基于图片内容的3个洞察或建议
```

### 场景6：商业文案与营销

```
为一款[产品类型]撰写小红书种草笔记矩阵：
- 主推笔记1篇（1500字，含5个标题备选）
- 评测笔记1篇（1000字，真实使用感）
- 问答笔记3篇（每篇300字，针对不同痛点）
目标人群：[描述]
调性：真实、生活化、不能有广告感
每篇附10个SEO标签
```

### 场景7：数据分析与可视化

```
以下是过去12个月的销售数据：
[粘贴数据]

请完成：
1. 按月/季度的趋势分析
2. 同比环比计算
3. 异常值识别及可能原因
4. 用Python生成可视化图表的代码（matplotlib）
5. 3条基于数据的业务建议
```

### 场景8：翻译与跨语言工作

```
请将以下学术摘要翻译为英文：
要求：
- 保持学术论文的正式语域
- 专业术语提供中英对照（首次出现时括号标注）
- 长句拆分为符合英文习惯的短句
- 被动语态优先
- 标注3处可能的表达优化建议

[中文摘要]
```

## Gemini 3.1 Pro vs GPT-5.4 vs Claude 4.6：怎么选？

不同模型有不同强项。实测总结：

| 场景 | 最佳选择 | 原因 |

**务实建议：** 不需要为每个模型单独注册账号。在[GPTCat](https://gptcat.cc)一个平台上切换使用所有模型，让每个任务都用最合适的AI。如果只需要Gemini+GPT的核心能力，[SnakeGPT](https://snakegpt.vip)也完全够用。

## 5个让Gemini效果翻倍的技巧

### 技巧1：给角色，不给指令

```
差：翻译这段话
好：你是一位拥有20年经验的同声传译专家，擅长中英法三语。
   请将以下商务谈判内容翻译为英文，保持谈判语气的微妙之处。
```

### 技巧2：用"分步执行"处理复杂任务

```
接下来我会给你一个复杂任务，请按以下步骤执行：
步骤1：先理解需求，用3句话复述确认
步骤2：列出完成任务需要的信息，缺什么问我
步骤3：生成大纲让我确认
步骤4：确认后再开始正式输出
```

### 技巧3：利用100万上下文做"知识库"

先一次性喂入大量参考资料，然后基于这些资料提问。Gemini会在你提供的资料范围内回答，大幅减少"幻觉"。

### 技巧4：多模型交叉验证

对于重要决策，用Gemini和GPT-5.4分别回答同一个问题，对比两者的分析角度。在GPTCat上一键切换即可实现。

### 技巧5：保存好用的Prompt

发现一个效果好的Prompt后，保存下来反复使用。建议建一个个人Prompt库，按场景分类。

## 常见问题FAQ

### Q：Gemini 3.1 Pro是免费的吗？
A：Google官方平台有免费额度但限制严格。通过GPTCat或SnakeGPT使用，注册免费，正常使用需要额度。具体方案见 [Gemini免费使用攻略](/guide/gemini-free-usage-complete-guide-2026)。

### Q：需要翻墙吗？
A：通过GPTCat或SnakeGPT使用**不需要翻墙**，国内网络直接访问。如果要用Google官方平台则需要。详见 [Gemini国内打不开怎么办](/guide/gemini-china-access-solution-2026-march)。

### Q：手机上能用吗？
A：可以。GPTCat和SnakeGPT都是网页版，手机浏览器直接打开就能用，iPhone和安卓都支持，不需要下载App。

### Q：Gemini的中文好吗？
A：Gemini 3.1 Pro的中文能力显著提升，日常对话和写作已经足够流畅。但极致的中文文学创作仍推荐DeepSeek-V3（在GPTCat可切换使用）。

### Q：和ChatGPT比，Gemini有什么优势？
A：最大优势是100万token上下文和多模态原生能力。处理长文档、视频分析、大型代码库这类场景，Gemini目前无可替代。详细对比见 [Gemini vs ChatGPT 2026全面对比](/guide/gemini-vs-chatgpt-2026-comparison)。

### Q：Gemini API怎么用？
A：开发者可以通过Google AI Studio获取API Key。详细教程见 [API Key免费申请指南](/guide/api-key)。

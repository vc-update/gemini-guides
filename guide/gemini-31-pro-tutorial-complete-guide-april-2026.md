## Gemini 3.1 Pro 有什么不同？核心能力速览

在深入教程之前，我们先了解 Gemini 3.1 Pro 相较于上一代的核心升级，这直接影响你的使用场景选择。

Gemini 3.1 Pro 支持文字、图像、音频、视频的混合输入，真正实现了"看图说话+听声辨意"的一体化处理。开发者社区反馈，该模型在处理跨文件代码库的长上下文任务时表现尤为突出——这也是 Cline 等编程工具选择深度集成 Gemini 3.x 系列的核心原因。

### 推理与数学能力跃升

在最新的国际基准测试中，Gemini 3.1 Pro 的数学推理和科学问答成绩超越了同期多数主流模型，尤其在多步骤逻辑链推导上有明显提升。

## Gemini 3.1 Pro 完整使用教程（分场景详解）

### 场景一：长文档分析与总结

**步骤：**
1. 在对话框中点击📎上传附件，支持 PDF、Word、TXT 等格式
2. 输入指令，例如：`请总结这份报告的核心论点，并列出数据支撑`
3. 模型将逐段理解全文并输出结构化摘要

**技巧：** 对于超长文档，建议分章节上传，并在提示词中注明"结合上下文理解"，输出质量更佳。

### 场景二：图像理解与多模态分析

1. 点击图像上传按钮，支持 JPG/PNG/WEBP
2. 可直接提问，例如：`这张图中的数据图表说明了什么趋势？`或`帮我识别图中的文字并翻译成中文`
3. Gemini 3.1 Pro 能够同时处理图文混合问题，支持多图对比分析

### 场景三：代码生成与调试

Gemini 3.1 Pro 在代码任务上的表现堪称一流，尤其适合以下场景：

- 输入需求说明，生成完整函数或模块
- 粘贴报错信息，快速定位 bug 原因
- 跨文件重构建议（需配合上下文粘贴）

**示例提示词：**
```
用 Python 写一个爬取京东商品价格的脚本，要求：
1. 使用 requests + BeautifulSoup
2. 输出商品名称、当前价格、链接
3. 加入异常处理和请求延迟
```

### 场景四：中文写作辅助

Gemini 3.1 Pro 的中文能力在 3.x 版本中得到大幅强化，以下任务均有良好表现：
- 公文写作、商业报告撰写
- 小红书/微信公众号文案生成
- 学术论文润色与降重建议
- 多轮对话式头脑风暴

## 常见问题解答（FAQ）

### Q：Gemini 3.1 Pro 和 Gemini 3.0 有什么区别？

3.1 Pro 在3.0基础上重点强化了长上下文稳定性、多轮对话一致性，以及代码和数学任务的准确率。整体可视为3.0的全面优化升级版。

### Q：国内使用 Gemini 3.1 Pro 需要谷歌账号吗？

如果通过 GPTCat 或 SnakeGPT 等聚合平台使用，**不需要**谷歌账号，国内邮箱注册即可直接使用。

### Q：Gemini 3.1 Pro 支持中文语音输入吗？

在 GPTCat 平台上，语音功能已完整还原，支持普通话语音输入，识别准确率较高。

### Q：上传文件有大小限制吗？

官方 API 限制单文件不超过 20MB，镜像平台的具体限制以各平台说明为准，建议超大文件分段处理。

## 使用技巧：让 Gemini 3.1 Pro 发挥最大效能

1. **善用系统提示词（System Prompt）：** 在对话开始前设置角色和输出格式，可显著提升回答质量
2. **明确输出结构：** 要求模型用"分点列举""表格对比""步骤编号"等方式输出，便于阅读
3. **多轮追问：** 不满意初始回答时，直接说"请更深入地展开第2点"，模型会补充细节
4. **温度参数调节：** 在创意写作任务时适当提高 temperature，在代码/事实类任务时降低，提升准确性
